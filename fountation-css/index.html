What is CSS

Cascading Style Sheets allows you to create great-looking web pages, but how does it work under the hood? This article explains what CSS is, with a simple syntax example, and also covers some key terms about the language.

Syntax

  • selector - as the name suggests it will select the elements matching expression
  • property - what is the specification you want to modify
  • value - what you want to apply on the element

Deferent Selectors

Tag selector - selects the elements having the matching tags 

h1 {
  color: blue;
}

Class selector - The CSS class selector matches elements based on the contents of their class attribute. 

.special {
  color: blue;
}

ID - The CSS ID selector matches an element based on the value of the element's id attribute. In order for the element to be selected, its id attribute must match exactly the value given in the selector. 

#special {
  color: blue;
}

Combine - you could also combine these into a selector list, by adding a comma between them. 

h1,
.special {
  color: blue;
}
